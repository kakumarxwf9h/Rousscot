# M1COO_3Hopital
TP3 COO Master 1

Auteurs: Cyril Ferlicot et Aurélien Rousseau

Introduction:
Ce TP a pour objectifs:
    - mettre en application les notions de factory, singleton, domaine, métier
    - créer une application de gestion d'un hopital

Sommaire:
1) les packages
2) cas d'utilisation

1) les packages
    - domaine -> classes du domaine
    - factory -> classes factory gérant les classes du domaine (exception pour Speciality)
    - exception -> classes d'exceptions
    - metier -> classes du métier , ici des actions

2) cas d'utilisation
Le fichier ressources/howTo.txt liste un ensemble de cas d'utilisation utilisables sur l'application
